# OrangeScada Node-RED driver

Node-RED nodes for describing OrangeScada universal driver hierarchy:

```text
Driver -> Node -> Device -> Tag
```

## Start

```sh
npm install
npm start
```

Open:

```text
http://localhost:1880
```

## Driver defaults

```text
Host: 192.168.0.102
Port: 8891
SSL: off
Driver UID: nodered
Password: empty
```
